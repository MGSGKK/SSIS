About Project Types While Creating SSIS Package
	We Have 3 Project Type Temaplates While Creating SSIS Project
		A) Integration Services Project
			This Is The One Which Can be Used to Implement the SSIS Packages From Scratch.
			
		B) Integration Services Project(Azure Enabled)
			This Is The One Which Can be Used to Implement the SSIS Packages From Scratch With Additional Functionality Which be Dealing With Azure Cloud.
			
		C) Integration Services Import Project Wizard
			Which Can Be Used to Import the Project from SQL Server Or From .ISPAC FIle File.
